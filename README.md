## About

*A Discord Server Cloner Which Can Clone Any Discord Server In Just Few Minutes.*
- Clone Channels
- Channel Permissions
- Server Roles
- Server Name
- Server Banner / Avatar
- Server Emojis
- Server Settings 

## Installation
```js
#install dependencies
$ pip install
build for production and launch server
$ python3 main.py
```
